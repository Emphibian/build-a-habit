# Build a Habit
A habit-tracker that allows you create habits and track them effectively. Habits may be generated daily or on a weekly basis.

## Running locally
First clone the repo. Then `cd` to the root.
## For the frontend:
`cd react`
`npm install`

After installation, run the:
`npm run dev`

## For the backend:
`cd server`
`npm install`
`npm run dev`

### MongoDB
You will have to specify a mongoDB URL.
One way to do it would be to set the environment variable `MONGODB_URL` before running the commands.
